# Roadmap

Planned features and improvements for ArabPy.

## Version 1.1.0 (Planned)

### Core Features
- [ ] Enhanced error messages in Arabic
- [ ] Improved tokenizer accuracy
- [ ] Better handling of edge cases
- [ ] Performance optimizations

### Dictionary Expansion
- [ ] NumPy module translations (500+ terms)
- [ ] Pandas module translations (300+ terms)
- [ ] Django framework translations (200+ terms)
- [ ] Flask framework translations (150+ terms)
- [ ] Requests library translations (50+ terms)

### Documentation
- [ ] Arabic documentation
- [ ] Video tutorials
- [ ] Interactive examples
- [ ] API reference improvements

## Version 1.2.0 (Planned)

### IDE Integration
- [ ] VS Code extension

  - Auto-completion for Arabic keywords
  - Inline translation hints
  - Error messages in Arabic

- [ ] PyCharm plugin
  - Arabic syntax support
  - Code completion
  - Translation tools

### Developer Tools
- [ ] Arabic REPL (read-eval-print loop)
- [ ] Translation CLI tool
- [ ] Code formatter for Arabic Python
- [ ] Linter for Arabic Python code

### Testing
- [ ] Expanded test coverage
- [ ] Integration tests
- [ ] Performance benchmarks
- [ ] Regression test suite
