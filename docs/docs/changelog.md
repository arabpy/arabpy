# Changelog

All notable changes to ArabPy will be documented in this file.

## [1.0.0] - 2024-01-15

### Added
- Initial release of ArabPy
- `ArabPyTranslator` class for bidirectional translation
- `arabic_code_runner` function for direct code execution
- Translation dictionary with 3,200+ entries
- Support for Python 3.8+
- Comprehensive test suite (92+ tests)
- Full type hints and documentation
- File translation support
- Translation statistics
- Round-trip translation capability

### Features
- Python to Arabic translation
- Arabic to Python translation
- String and comment preservation
- Indentation preservation
- UTF-8 support
- Error handling with traceback preservation
- Tokenizer-based accurate translation

### Documentation
- Complete API reference
- Comprehensive examples
- Quick start guide
- Installation instructions
- FAQ section

## [Unreleased]

### Planned
- NumPy module translations
- Pandas module translations
- VS Code extension
- Arabic error messages
- Performance optimizations
- Expanded dictionary coverage

## Version Format

We follow [Semantic Versioning](https://semver.org/):

- **MAJOR**: Incompatible API changes
- **MINOR**: Backwards-compatible functionality additions
- **PATCH**: Backwards-compatible bug fixes

## Categories

### Added

### Changed

### Deprecated

### Removed

### Fixed
